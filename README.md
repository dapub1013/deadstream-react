# DeadStream React

A dedicated Raspberry Pi device for streaming Grateful Dead concerts from the Internet Archive.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Dexie.js (IndexedDB)
- HTML5 Audio API

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Run tests
npm run test

# Build for production
npm run build
```
